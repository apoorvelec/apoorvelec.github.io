This page gives a brief description of the language that is being developed in this project. Refer to this page regarding the syntax of the language being developed.

  • Language Name:


    J++

  • Syntax


    • Primitives: int, double, string, char, short, long, byte, bool
    • for loop: Exactly the same as in Java. For eg:
      					for(int i=0;i<10;i++){
						//statements here
					}
    • if/else block: Exactly the same as in Java. For eg:
      						if(//true or false condition){
							// some statements go here
						}else{
							// another set of statements
						}
    • Comments: See below example:
      						// This is a single line statement
						/*This is a multiline 
						  statement which spans multiple lines.*/
    • Functions/Methods: Methods will be similar to Java but very simple. They can accept the primitives as their argument (any number of args) . They can have a return type(corresponding to each primitive) or if they don't return anything, the return type is void. For eg,
      						// This is a method
						int sum(int a, int b){
							return (a+b);
						}
